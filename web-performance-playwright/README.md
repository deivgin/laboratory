# Web vitals performance testing

This project is dedicated to learning more about and exploring web performance testing using the playwright testing framework. Core goal of this project is to better understand what metrics are important, how they are calculated and how we can implement this testing in a playwright test suit.

The project is separated into three parts:

1. A simple express server to send data to the frontent
2. React client to render the data
3. Playwright project to run performance tests
